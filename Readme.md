#### Documentação para consultar a API


GET ``` area/{nomeDaCasa} ``` <br/>

GET ``` valor/{nomeDaCasa} ``` <br/>

GET ``` areaComodos/{nomeDaCasa} ``` <br/>

GET ``` maiorComodo/{nomeDaCasa} ``` <br/>

POST ``` imovel```

Exemplo de body:

```
   {
    "name": "Novacasa",
    "district": {
        "name": "Bairro do Limoeiro",
        "squaredMeterValue": 200
    },
    "rooms": [
        {
            "name": "Quarto",
            "width": 10,
            "length": 20
        }
      ]
    }
```
